# 🏛️ Student Academic Platform (Institutional Command Center)

A high-density, AI-powered academic intelligence dashboard designed for institutional scale. This platform currently manages **8,008 student records** with absolute mathematical precision and real-time analytical mapping.

## 🚀 Core Features

- **Institutional Analytics Hub**: Real-time KPI mapping of **Avg CGPA (7.95)**, **Institutional DNA Score (71)**, and **Placement Readiness (77.46%)**.
- **AI Weekly Insight Engine**: Automated pedagogical recommendations and strategic institutional forecasting.
- **Micro-Segmentation**: Advanced student clustering (High Achievers, Stable Performers, Improving, and Critical Zone) across all departments.
- **Departmental Intelligence**: Deep-dive analytics for CSE, AIML, IT, ECE, EEE, and MECH.
- **Predictive Engine**: AI-driven CGPA forecasting and student risk-probability mapping.

## 🛠️ Technology Stack

- **Frontend**: Next.js 14+ (Turbopack Enabled), TailwindCSS, Chart.js, Lucide Icons.
- **Backend**: FastAPI (Python 3.10+), SQLAlchemy Core, SQLite (WAL Enabled).
- **Security**: Institutional Retinal-Sync Authentication, JWT State Management.
- **Performance**: High-Speed analytical dataframe processing (Pandas/Scikit-learn).

## ⚙️ Installation & Deployment

### Backend Synchronization
1. `cd server`
2. `python -m venv venv`
3. `source venv/bin/activate` or `.\venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. `uvicorn app.main:app --reload`

### Frontend Initalization
1. `cd client`
2. `npm install`
3. `npm run dev`
